## Run Instructions:

### Step 1:
If you are not already in, run this line in your terminal to ensure you are in the correct folder (contact-app):
``` 
      cd .\contact-app\
```

### Step 2:
Make sure you have npm installed, run this line in your terminal
```
			npm install
```

### Step 3:
Put this line into your terminal and hit 'Enter' 
```
			npm run dev
```
